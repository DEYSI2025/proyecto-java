# Proyecto Java Web – CRUD Empleados

Proyecto desarrollado en Java Web (Servlets + JSP) que permite:

- Login de usuario
- CRUD de empleados
- Conexión a MySQL
- Generación de reporte PDF
- Búsqueda en tiempo real

## Tecnologías
- Java 21
- JSP / Servlets
- MySQL
- Apache Tomcat 10
- OpenPDF
